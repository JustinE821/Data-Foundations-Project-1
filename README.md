# Data-Foundations-Project

In order to run the project, first create a virual environment using 

py -m venv venv

Then, activate the environment using

./venv/Scripts/activate

Next, install dependencies using 

pip install -r requirements.txt

Now you can download the csv files from the links below and place them in /data. 

Finally, run the data processing pipeline with

py src/main.py

## Dataset links
New York Wildfires - https://catalog.data.gov/dataset/new-york-state-forest-ranger-wildland-fire-reporting-database-beginning-2008

California Wildfires - https://catalog.data.gov/dataset/california-fire-perimeters-all-5b4ee

Oregon Wildfires - https://catalog.data.gov/dataset/odf-fire-occurrence-data-2000-2022

National Wildfires - https://catalog.data.gov/dataset/national-interagency-fire-occurrence-sixth-edition-1992-2020-feature-layer
